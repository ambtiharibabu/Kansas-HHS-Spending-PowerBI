
# Kansas HHS Spending - Power BI Dashboard Portfolio

Federal spending analysis on U.S. Department of Health and Human Services (HHS) contracts and grants in Kansas from 2020 to 2024, built as a two-phase Power BI dashboard project.

**Data Source:** [USAspending.gov](https://www.usaspending.gov) — official U.S. government open data portal  
**Dataset:** 12,542 records | 2020–2024 | Grants + Contracts | Kansas recipients  
**Tools:** Power BI Desktop, DAX, Power Query  

---

## Project Structure

| Folder | Description |
|---|---|
| `project1-classic/` | Classic dashboard — KPI cards, bar/column/donut charts, matrix, trend line |
| `project2-advanced/` | Advanced dashboard — multi-line KPI cards, Funnel chart, Sankey diagram |

---

## Key Insights Across Both Projects

- Kansas received approximately **\$22.56 billion** in total HHS funding over 5 years
- **97.1% of funds came through Grants**, with Contracts making up only 2.9%
- Spending peaked in 2021 at **\$5.04B** (post-COVID surge), then stabilized near \$4.5B annually
- **Shawnee County** led all counties with \$20B in total spending and \$5.67M average award
- **Kansas City** was the top-funded city at \$584M, followed by Lenexa at \$450M

## Skills Demonstrated

- Power BI dashboard design (3x2 grid layout, storytelling flow)
- DAX measures: YoY Growth %, Total Spending, Recipient Avg Award, Distinct Recipients

- Advanced visuals: Sankey diagram, Funnel chart, conditional formatting matrix
